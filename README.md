# Moresampler semi-auto otoing pipeline

This is a simple notebook that helps you set up an environment on Google colab for you to auto oto your voicebanks. I don't actually own any part of the software used in the notebook tho.

# How to use it

1. Download this repo / clone it / extract it if it's a zip
2. Go to (Google Colab)[https://colab.research.google.com/]
3. Look at the top of your screen, find the tab "File"

4. Click it and scroll down, click open notebook
5. Click upload the notebook file you've just downloaded (.ipynb)
6. Follow the instructions inside
